# 今日热榜

数据来源：[tophub.today/topics](https://tophub.today/topics)，汇聚全网热门榜单，将相似新闻聚合为话题。

## 使用

```bash
uv run tophub.py
```

输出 JSON 到标准输出，每条为一个热点话题。**不要将输出保存为文件**，直接读取标准输出使用即可：

```json
[
    {
        "title": "苹果发布 MacBook Neo：搭载 A18 Pro 芯片，起售价 4599 元",
        "desc": "苹果在特别体验活动中推出全新产品 MacBook Neo，搭载 A18 Pro 芯片，定位入门级，是目前最便宜的 MacBook，起售价 4,599 元。外观延续铝合金外壳设计，采用柔和圆角，颜色丰富，无风扇设计可静音运行。提供教育优惠，叠加后国行版可能 4,000 元左右。其硬件参数包括 6 核 CPU、5 核 GPU、16 核神经网络引擎，13 英寸显示屏，8GB 内存 + 256GB 固态硬盘，最长 16 小时续航等，部分配置可选配 Touch ID 指纹识别，有红粉、靛蓝等多种颜色，整体重量 1.22 千克。",
        "time": "15小时前",
        "hot": 28
    }
]
```

字段说明：
* `title` 话题标题
* `desc` 话题摘要
* `time` 最新报道的发布时间
* `hot` 该话题的报道数量，数值越高关注度越高
